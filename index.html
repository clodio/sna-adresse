Géocodage inversé

A partir d'une coordonnée géographique retrouvez les adresses

GET 'https://api.laposte.fr/address/v1/reverse-geocoding/'

Les paramètres lat et lon sont obligatoires:

GET 'https://api.laposte.fr/address/v1/reverse-geocoding/?lon=2.37&lat=48.357'

Le paramètre type permet forcer le type de retour:

GET 'https://api.laposte.fr/address/v1/reverse-geocoding/?lon=2.37&lat=48.357&type=street'

Le retour est au format blablabla
